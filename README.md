# PatiYuva
Bu proje, hayvan sahiplendirme sürecini dijital ortamda güvenli ve düzenli şekilde yürütmek amacıyla geliştirilmiş bir web uygulamasıdır. Hayvan sahipleri, sahiplendirmek istedikleri hayvanları sisteme ekleyebilir; sahiplenmek isteyen kullanıcılar ise bu hayvanları inceleyip sahiplenme talebi oluşturabilir.
